Mon Projet CI/CD

Exemple simple pour démontrer un pipeline & captures d'écran

Présentation

Cette page est un exemple minimaliste en HTML & CSS. Tu peux l'utiliser pour montrer un dépôt, lancer des tests automatiques et capturer le déploiement.

Build

Étape : compilation / préparation

Test

Étape : exécution des tests automatisés

Deploy

Étape : déploiement sur serveur / hébergement

Comment utiliser

  1. Place ces fichiers dans ton dépôt Git (index.html & css/styles.css).
  2. Push sur GitHub / GitLab et lance ton pipeline CI/CD.
  3. Fais des captures : page d'accueil, logs du pipeline, etc.
